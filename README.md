# Udemy Clone

A responsive single-page clone of the Udemy homepage built using **HTML5** and **CSS3**. This project was created to practice front-end web development by recreating the layout and design of the Udemy website.

## Features

- Responsive single-page layout
- Navigation bar with search option
- Course categories section
- Course cards with images, ratings, and prices
- Clean and modern UI
- Footer section

## Technologies Used

- HTML5
- CSS3

## Purpose

This project was developed to improve my front-end development skills, including HTML structure, CSS styling, responsive design, Flexbox, and page layout techniques.

```
├── index.html
├── style.css

```
